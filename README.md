## Descripcion
Desarrollando un sistema base para la gestion de un hospital.
## Objetivo
Este proyecto busca implementar un sistema hospitalario en Java para aplicar conceptos POO , como clases abstractas, herencia, polimorfismo, encapsulamiento y  paquetes.

## Estructura de paquetes
- modelo.abstractas
- modelo.personas
- modelo.hospital
- modelo.enums
- servicios
- principal

## Clases principales
- Persona
- Empleado
- Paciente
- Medico
- Cirujano
- Enfermero
- Hospital
- CitaMedica
- Diagnostico
- Especialidad

## Enums
- EstadoCita
- Turno

## Historias de usuario
| ID | Historia | Prioridad | Story Points |
|---|---|---|---|
| HU-01 | Como recepcionista, quiero registrar pacientes para mantener un historial medico digital. | Alta | 5 |
| HU-02 | Como administrador, quiero contratar empleados para gestionar el personal del hospital. | Alta | 5 |
| HU-03 | Como recepcionista, quiero agendar citas validando la disponibilidad del medico. | Alta | 8 |
| HU-04 | Como medico, quiero registrar diagnósticos y recetas para llevar control de la atencion. | Alta | 5 |

## Conceptos aplicados
En este trabajo se están implementando los siguientes conceptos:
- clases abstractas
- herencia
- polimorfismo
- encapsulamiento
- uso de getters y setters
- relaciones entre clases
- uso de enums
